## 🔧 技术问题定位

### 问题根源
服务器端 `vtext_server/app.py:84-91` zstd 解压代码：
```python
dctx = zstd.ZstdDecompressor()
tmp.write(dctx.decompress(content))  # ← 大文件一次性解压内存溢出
```

### 验证方法
1. 临时禁用客户端压缩（COMPRESS_THRESHOLD=999999MB）
2. 重跑失败视频 → 17/70 成功
3. 结论：问题 100% 在服务器端，与 whisper.cpp 无关

### 建议修复
服务器端改用流式解压：
```python
dctx = zstd.ZstdDecompressor()
with dctx.stream_reader(content) as reader:
    tmp.write(reader.read())  # 分块读取
```
